About me

All about my passion

My name is Cleveland Williams and I am a web developer who specializes in creating dynamic and beautiful websites.Before my journey as a web developer I repaired computers & build high end gaming pcs. After building computersfor so long, I became interested in web development after which I encountered valuable resources. Among them wereCoding Dojo an onsite bootcamp, Udemy and FreeCodeCamp.

The Coding Dojo courses were focused on laying a foundation for web development. The instructors were awesome andvery knowledgeable. I started out with the fundamentals of HTML, CSS and JavaScript. The instructors taught us threefull stacks including Python, M.E.A.N and Ruby on Rails. After the first few weeks of building cool web apps withdifferent frameworks, my love affair with web development began. I would literally spend hours working on projectswhile implimenting new features and techniques. Web development is now my obsession and without it feels like a partof me is missing.

Process of creation

All about my work


Planning

Create an outline of the site content along with a functional flow chart. Document programmatic flow and gather generalinformation including samples, style guides, images and graphic files.


Design

Create color layouts based on approved ‘wireframe’ layouts. Research stock photography and develop graphics. Create styleguide of colors and fonts for site development.


Content Writing

Content writing and compiling usually overlaps with other stages of website creation, and its role can’t be underestimated.Content writing involves creation of headlines, text editing, writing new text, compiling the existing text, etc...


coding

The home page is usually created first, and then all sub-pages are added, according to the website hierarchy thatwas previously created in the form of a sitemap. All static page elements that were designed during the layout creationwill be created and tested.
